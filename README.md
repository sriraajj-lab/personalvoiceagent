# Aria Voice Agent

Personal AI voice co-pilot that takes over calls and speaks in your voice.

## Setup
1. Clone this repo
2. Run `npm install`
3. Add your ElevenLabs API key and agent ID to an `.env` file
4. Run `npx react-native run-android`

## Features
- ElevenLabs cloned voice
- Knowledge base with your profile
- Voice Activity Detection for natural conversations
- Background call handling
- Trigger word activation
